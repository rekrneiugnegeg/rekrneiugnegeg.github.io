Немного о JavaScript

JavaScript это язык программирования, применяемый в основном для исполнения в браузере. С помощью него вы можете придать своему сайту больше интерактивности и решать такие задачи как

  • Регистрация и авторизация пользователей
  • Получение обновлений информации без пепрезагрузки страницы
  • Обработка информации пользователя
  • Показывать подсказки при наведении указателя
И это далеко не полный список. А ещё используя некоторые фреймворки можно использовать JavaScript не только в браузере, но при создании бэкенда, desktop и мобильных приложений. Таким образом, JavaScript это один из наиболее важных языков программирования современности.
